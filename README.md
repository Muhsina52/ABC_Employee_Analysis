# ABC Company Employee Analysis

## Project Overview

This project analyzes the ABC Company employee dataset using Python. The project focuses on data preprocessing, exploratory data analysis (EDA), visualization, and business insights.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook (VS Code)

## Project Tasks

- Preprocessed the dataset by replacing the Height column with random values between 150 cm and 180 cm.
- Analyzed employee distribution across teams and positions.
- Identified the predominant age group.
- Determined the team and position with the highest salary expenditure.
- Analyzed the correlation between Age and Salary.
- Created visualizations to present the findings.

## Key Insights

- The New Orleans Pelicans team has the highest number of employees.
- SG (Shooting Guard) is the most common position.
- Most employees belong to the 20–29 age group.
- Cleveland Cavaliers has the highest salary expenditure.
- The Center (C) position has the highest salary expenditure.
- There is a weak positive correlation between Age and Salary.
